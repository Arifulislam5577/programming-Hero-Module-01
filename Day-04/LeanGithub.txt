1)How to see all branch?

--> To see all branches with command "git branch".

2)How to create a new branch?

--> Create a new branch with command "git branch [branch name]".

3)How to switch one branch to another?

--> Switch one branch to another with command "git checkout [branch name]]"


4)How to merge branch?

--> Merge one branch to another with command "git merge [branch name]"

5)How to create a new branch and switch to this branch?

--> Create a new branch and switch to this branch with command "git checkout -b [branch name]"
